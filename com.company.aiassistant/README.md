**AI Copilot for Eclipse IDE**!

This AI-powered assistant integrates directly into Eclipse, helping developers understand, generate, and improve code without leaving their development environment.

### 🔧 Flexible AI Backend

* Runs with **Ollama** out of the box
* Supports **any Ollama-compatible model**
* Connects to **custom AI models and APIs**
* Easily switch between **local and cloud-based LLMs**

### ✨ Key Features

🧠 **Explain Code** – Understand complex code instantly.

🐞 **Fix Bugs** – Detect and apply fixes directly to source files with a single click.

⚡ **Improve Code** – AI-powered refactoring and code optimization applied directly in the editor.

💻 **Generate Code** – Create methods, classes, and implementations from natural language prompts.

🏗️ **Generate Complete Projects** – Scaffold full project structures and starter code in seconds.

💬 **AI Chat** – Get coding help, architecture guidance, and technical insights without leaving Eclipse.

The goal is to bring modern AI-assisted development to Eclipse while keeping the platform flexible, model-agnostic, and developer-friendly.
